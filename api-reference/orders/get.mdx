---
title: 'Get order'
description: 'Retrieve a single order by its numeric ID. Authentication required.'
api: 'GET http://localhost:1337/api/orders/{id}'
---

Returns a single order by its numeric ID. Authentication is required.

<Warning>
  This endpoint requires authentication. Requests without a valid `Authorization: Bearer <jwt_token>` header will return a `401 Unauthorized` error.
</Warning>

## Path parameters

<ParamField path="id" type="number" required>
  Numeric ID of the order to retrieve.
</ParamField>

## Order states

The `state` field reflects the current stage of the order lifecycle:

| State | Description |
|---|---|
| `Pedido creado` | Order has been created and payment is pending. |
| `Pago confirmado` | Payment was confirmed by the gateway. |
| `Preparando tu café` | Order is being prepared. |
| `Enviado` | Order has been shipped. |
| `Entregado` | Order was delivered to the customer. |
| `Cancelado` | Order was cancelled. |

## Response

<ResponseField name="data" type="object">
  The order object.

  <Expandable title="properties" defaultOpen={true}>
    <ResponseField name="id" type="number">
      Unique numeric identifier.
    </ResponseField>

    <ResponseField name="totalPayment" type="number">
      Total amount charged, calculated from product prices after discounts.
    </ResponseField>

    <ResponseField name="idPayment" type="string">
      UUID payment reference used with the Bold payment gateway.
    </ResponseField>

    <ResponseField name="state" type="string">
      Current order status. See the state table above for possible values.
    </ResponseField>

    <ResponseField name="addressShiping" type="object">
      JSON shipping address object provided at checkout.
    </ResponseField>

    <ResponseField name="products" type="object[]">
      JSON array of products in the order.
    </ResponseField>

    <ResponseField name="user" type="object">
      Associated user. Populated when `populate=user` is included.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl "http://localhost:1337/api/orders/7?populate=*" \
  -H "Authorization: Bearer <jwt_token>" \
  -H "Content-Type: application/json"
```

</RequestExample>
